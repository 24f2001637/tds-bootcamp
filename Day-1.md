--- After Day-1 ---

I learned these things as well, apart from the checklist:
* How absolute paths (starting with `/`) differ from relative paths, and how `pwd` helps identify the current location.
* That hidden files in Linux begin with a dot (`.`) and require `ls -a` to be viewed.
* How to use the `echo` command to print text and pipe it directly into a file using redirection.
* The difference between `sudo apt update` (which only checks for updates) and `sudo apt install` (which actually installs packages like Git).
